## Lem_inVis
Каркас для визуализатора движения муравьев по карте (задание из школы 21).
# Пример ввода:

```
5
##start
1 23 3
2 16 7
3 16 3
4 16 5
5 9 3
6 1 5
7 4 8
##end
0 9 5
1-2
1-3
2-4
2-5
2-7
3-4
3-5
4-0
4-7
5-6
6-0
6-7

L1-2 L2-3
L1-4 L3-2 L2-5 L4-3
L1-0 L3-4 L5-2 L2-6 L4-5
L3-0 L5-4 L2-0 L4-6
L5-0 L4-0
```
